# V7模型最终版本 - 跨文化适应研究完整文档

## 📋 项目概述

本项目包含**香港样本**（N=75）和**法国样本**（N=249）的完整V7跨文化适应预测模型，采用5折交叉验证、DeepFM深度学习和Stacking集成学习方法，确保跨文化研究的方法论一致性。

### 香港样本性能
- **测试集R² = 0.7534**（超过线性回归11.8%）
- **5折交叉验证R² = 0.7267 ± 0.0011**（极其稳定）
- **RMSE = 1.9981，MAE = 1.5367**

### 法国样本性能
- **测试集R² = 0.7558**（超过线性回归18.0%）
- **5折交叉验证R² = 0.7558 ± 0.0038**（稳定）
- **RMSE = 2.0757，MAE = 1.6440**

## 🔬 理论贡献

### 1. 跨文化适应的文化差异
- **法国华人**：文化接触是核心（0.944），反映了融入西方主流社会的挑战
- **香港新移民**：社会联结感最重要（0.733），反映了建立社会网络的需求

### 2. 家庭支持的文化特异性
- 在法国样本中重要性显著更高（0.748 vs 0.336）
- 可能反映了远离祖国后对家庭联系的更强依赖

### 3. 时间因素的差异化影响
- 来法时长影响更大（0.617 vs 0.129）
- 可能与语言障碍、文化距离等因素相关
