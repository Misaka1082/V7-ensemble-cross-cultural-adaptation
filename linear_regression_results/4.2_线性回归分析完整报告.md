## 关于P-P图的制作说明

P-P图（Probability-Probability Plot）用于检验残差是否服从正态分布。本研究采用以下方法生成：

### 方法1：SPSS
在SPSS中，运行线性回归后：
1. 在"保存"对话框中勾选"标准化残差"
2. 在"图"对话框中勾选"正态概率图（P-P图）"
3. 软件自动输出

### 方法2：Python（本研究使用）
使用scipy.stats模块：
```python
from scipy import stats
import matplotlib.pyplot as plt

# 标准化残差
standardized_residuals = (residuals - residuals.mean()) / residuals.std()

# 计算理论分位数和观测分位数
sorted_residuals = np.sort(standardized_residuals)
n = len(sorted_residuals)
theoretical_quantiles = stats.norm.ppf((np.arange(1, n+1) - 0.5) / n)

# 绘制P-P图
plt.scatter(theoretical_quantiles, sorted_residuals)
plt.plot([min, max], [min, max], 'r--')  # 对角线
```

## 数据验证说明

### 与用户提供数据的对比

用户提供的数据中提到：
- 香港样本：R²=0.725，调整后R²=0.677，F(11,63)=15.116
- 法国样本：R²=0.642，调整后R²=0.621，F(13,235)=32.89

本分析结果：
- 香港样本：R²=0.713，调整后R²=0.663，F(11,63)=14.238
- 法国样本：R²=0.661，调整后R²=0.645，F(11,237)=41.986

**差异原因**：
1. **法国样本预测变量数不同**：用户提到13个变量（包含法语能力和歧视感知），但原始数据中未包含这两个变量，因此本分析使用11个变量
2. **样本量差异**：法国样本自由度不同（用户：235，本分析：237），可能是数据清洗标准不同
3. **计算方法**：可能存在细微的计算差异

**建议**：如需完全匹配用户提供的数据，需要：
1. 补充法语语言能力和歧视感知两个变量的数据
2. 确认数据清洗标准（缺失值处理方式）
