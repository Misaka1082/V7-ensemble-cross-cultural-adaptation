## Simulated Data Quality Validation Details

### D.4 Baseline Model R²

A linear regression using only main effects yielded an R² of .713 on the real data and .711 on the simulated data (difference = .002). This negligible difference confirms that the simulated data preserves the predictive structure of the original sample and is suitable for interaction analysis.
